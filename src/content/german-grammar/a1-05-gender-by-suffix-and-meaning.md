## Ziel dieses Kapitels

Das Genus jedes deutschen Nomens zu lernen, wirkt anfangs mühsam. Mit **Endungs- und Bedeutungsregeln** erkennst du viele neue Wörter sofort richtig.

Am Ende kannst du:

- Genus anhand **häufiger Endungen** vorhersagen
- **Bedeutungsgruppen** für der, die, das nutzen
- typische **Ausnahmen** kennen
- neue Nomen **mit Artikel** lernen

---

# 1. Warum Endungen helfen

Deutsches Genus folgt selten der Logik. Aber **Endungen** sind ein starkes Signal:

| Endung | Genus | Beispiel |
| --- | --- | --- |
| -ung | die (≈95 %) | die Prüfung, die Rechnung, die Bedeutung |
| -heit / -keit | die | die Freiheit, die Möglichkeit |
| -schaft | die | die Freundschaft, die Mannschaft |
| -tion / -sion | die | die Information, die Diskussion |
| -ei | die | die Bäckerei, die Datei |
| -in (Person) | die | die Lehrerin, die Kollegin |
| -chen / -lein | das | das Mädchen, das Büchlein |
| -ment | das | das Dokument, das Apartment |
| -um | das | das Museum, das Datum |
| -nis | das | das Ergebnis, das Zeugnis |

**Merksatz:** *-ung* ist fast immer **die**. Das ist eine der zuverlässigsten Regeln.

---

# 2. Maskulin — der

| Muster | Beispiel |
| --- | --- |
| Tage, Monate, Jahreszeiten | der Montag, der Januar, der Sommer |
| Wetter (oft) | der Regen, der Schnee, der Wind |
| männliche Personen | der Lehrer, der Arzt, der Kollege |
| Endung -er (Berufe, Geräte) | der Computer, der Lehrer, der Drucker |
| Endung -ig / -ling | der König, der Schmetterling |
| Endung -ismus | der Tourismus, der Optimismus |
| Verben → Nomen (Agens) | der Fahrer, der Lehrer |

> **der** Dienstag, **der** Monitor, **der** Frühling

---

# 3. Feminin — die

| Muster | Beispiel |
| --- | --- |
| weibliche Personen | die Lehrerin, die Ärztin |
| Endung -ung | die Wohnung, die Prüfung |
| Endung -heit / -keit | die Gesundheit, die Schwierigkeit |
| Endung -ei | die Partei, die Bäckerei |
| Endung -schaft | die Landschaft |
| Endung -ik | die Musik, die Logik |
| Schiffe und Flugzeuge | die Titanic, die Lufthansa |
| viele Bäume und Blumen | die Eiche, die Rose |

> **die** Anwendung, **die** Aufgabe, **die** Information

---

# 4. Neutral — das

| Muster | Beispiel |
| --- | --- |
| Endung -chen / -lein | das Mädchen, das Häuschen |
| Infinitiv als Nomen | das Lernen, das Schreiben, das Essen |
| Endung -ment | das Instrument, das Segment |
| Endung -um | das Zentrum, das Forum |
| Endung -o (Loanwords) | das Kino, das Büro |
| junge Personen / Tiere | das Kind, das Lamm |
| Metalle und chemische Elemente | das Gold, das Eisen |

> **das** Mädchen, **das** System, **das** Dokument

**Wichtig:** *das Mädchen* — grammatisch neutral, obwohl es eine weibliche Person bezeichnet.

---

# 5. Bedeutung statt Endung

Manche Gruppen haben eine **Tendenz**:

| Bedeutung | Tendenz | Beispiel |
| --- | --- | --- |
| alkoholische Getränke | maskulin | der Wein, der Kaffee, der Tee |
| nichtalkoholische Getränke | neutral | das Wasser, das Bier (Ausnahme!) |
| Autos (umgangssprachlich) | maskulin | der VW, der BMW |

Diese Regeln sind **Hilfen**, keine Garantien. Bei Zweifel: **Wörterbuch** oder **mit Artikel lernen**.

---

# 6. Ausnahmen — die du kennen solltest

| Wort | Erwartung | Tatsächlich |
| --- | --- | --- |
| die E-Mail | -ung? | die E-Mail |
| der Urlaub | -ung? | **der** Urlaub |
| die Butter | -er? | **die** Butter |
| das Fenster | -er? | **das** Fenster |
| der See (See) | — | der See / **der** See (der See = lake) vs. die See (sea) |

Lerne Ausnahmen als **fertige Paare**: der Urlaub, die Butter, das Fenster.

---

# 7. Strategie für neue Wörter

1. **Endung prüfen** — passt eine Regel?
2. **Bedeutungsgruppe** — Person, Ding, Abstraktion?
3. **Im Kontext hören** — der/die/das im Satz
4. **Immer mit Artikel** in Vokabellisten notieren

> ~~Problem~~ → **das** Problem
> ~~Anwendung~~ → **die** Anwendung

---

# 8. Häufige Fehler

## Fehler 1 — Logik statt Grammatik

Falsch:

> die Mädchen (weil weiblich)

Richtig:

> **das** Mädchen

## Fehler 2 — Englisches Genus übertragen

Falsch:

> die Problem (problem = neuter in English thinking)

Richtig:

> **das** Problem

## Fehler 3 — Nur die deutsche Übersetzung lernen

Falsch:

> Anwendung, Prüfung, System

Richtig:

> **die** Anwendung, **die** Prüfung, **das** System

---

# 9. Vergleich mit Englisch

Englisch hat kein grammatisches Genus. *the application, the system, the problem* — alles **the**.

Deutsch braucht **der / die / das** — und die Endung beeinflusst später Adjektive und Pronomen.

---

# 10. Typische Fehler für vietnamesischsprachige Lernende

- Vietnamesisch hat **kein Genus** — trainiere Artikel **laut** beim Lernen.
- **Alle neuen Nomen mit Farbe oder Symbol** markieren: der = blau, die = rot, das = grün (persönliche Eselsbrücke).
- **Nicht raten** — lieber einmal nachschlagen als falsch einprägen.

---

# 11. Deutsch im Alltag

> die Wohnung ist klein.
> der Supermarkt ist um die Ecke.
> das Restaurant hat heute geöffnet.
> die Rechnung, bitte!
> der Termin ist am Montag.

---

# 12. Deutsch für Software Engineers

> die Anwendung startet nicht.
> das System ist offline.
> der Server antwortet nicht.
> die Datei ist zu groß.
> das Update dauert zehn Minuten.

# 13. Übungen

## Aufgabe 1 — Endung → Genus

Welcher Artikel passt?

a) ______ Prüfung

b) ______ Mädchen

c) ______ Computer

Lösung:

> a) **die** (-ung) — b) **das** (-chen) — c) **der** (-er)

## Aufgabe 2 — die, der oder das?

a) ______ Information

b) ______ Dokument

c) ______ Freiheit

Lösung:

> a) **die** (-tion) — b) **das** (-ment) — c) **die** (-heit)

## Aufgabe 3 — Ausnahme erkennen

Welches Wort bricht die -ung-Regel?

a) die Wohnung

b) der Urlaub

c) die Rechnung

Lösung:

> **b) der Urlaub** — Ausnahme: maskulin trotz -ung

## Aufgabe 4 — Lückentext

Ergänze: ______ Lernen macht Spaß.

a) Das

b) Die

c) Der

Lösung:

> **a) Das** — Infinitiv als Nomen ist neutral

## Aufgabe 5 — Neue Wörter raten

Rate das Genus (dann nachschlagen!):

a) die Konfiguration

b) das Backup

c) der Fehler

Lösung:

> a) **die** (-tion) — b) **das** (Lehnwort, neutral) — c) **der** (-er)

## Aufgabe 6 — Fehler korrigieren

Korrigiere:

> die Mädchen spielt im Park.

Mögliche Lösung:

> **Das** Mädchen **spielt** im Park.

## Aufgabe 7 — Vokabelliste

Schreibe fünf neue Nomen aus deinem Fachgebiet — **immer mit Artikel**.

## Aufgabe 8 — Schnelltest

Ordne zu: der / die / das

> Freundschaft, Museum, Montag, Ergebnis, Lehrerin

Lösung:

> die Freundschaft, das Museum, der Montag, das Ergebnis, die Lehrerin
