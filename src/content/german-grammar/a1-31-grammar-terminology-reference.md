## Ziel dieser Referenz

Schnelle **Nachschlagetabelle** für alle wichtigen **Grammatikbegriffe** auf A1-Niveau — Deutsch mit englischer und vietnamesischer Orientierung.

---

# 1. Wortarten — Wortarten

| Deutsch | Englisch | Beispiel |
| --- | --- | --- |
| das Nomen / Substantiv | noun | der Kurs |
| das Verb | verb | lernen |
| das Adjektiv | adjective | gut |
| das Pronomen | pronoun | ich, mein |
| der Artikel | article | der, ein, kein |
| die Präposition | preposition | in, für, mit |
| die Konjunktion | conjunction | und, aber, weil |
| das Adverb | adverb | heute, gern |

---

# 2. Satzglieder — Satzglieder

| Deutsch | Englisch | Frage |
| --- | --- | --- |
| das Subjekt | subject | Wer? Was? |
| das Prädikat | predicate | Was passiert? |
| das Objekt | object | Wen? Was? |
| die Ergänzung | complement | verschiedene |
| die Angabe | adverbial | Wann? Wo? Wie? |

---

# 3. Kasus — Fälle (A1)

| Kasus | Frage | A1-Verwendung |
| --- | --- | --- |
| der Nominativ | Wer? Was? | Subjekt, nach sein |
| der Akkusativ | Wen? Was? | direktes Objekt, für, ohne, um |

**Artikel Akkusativ maskulin:** der → **den**, ein → **einen**

---

# 4. Genus und Numerus

| Begriff | Werte |
| --- | --- |
| das Genus (Geschlecht) | maskulin (der), feminin (die), neutral (das) |
| der Numerus (Zahl) | Singular, Plural (immer **die**) |
| die Person | ich, du, er/sie/es, wir, ihr, sie |
| der Plural | die Tage, die Bücher, die Fragen |

---

# 5. Verben — Formen

| Begriff | Beispiel |
| --- | --- |
| der Infinitiv | lernen |
| das finite Verb | ich lerne |
| das Partizip II (A2) | gelernt |
| regelmäßig | machen → ich mache |
| unregelmäßig | sein → ich bin |
| Stammvokalwechsel | lesen → er liest |
| das Modalverb | können, müssen, wollen … |
| der Imperativ | Komm! / Kommen Sie! |

# 14. Eselsbrücken

| --- | --- |
