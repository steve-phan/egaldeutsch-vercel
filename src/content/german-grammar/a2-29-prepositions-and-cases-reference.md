## Ziel dieses Kapitels

Diese **Referenz** fasst alle **A2-Präpositionen** mit ihrem **Kasus** zusammen.

Am Ende kannst du:

- **Akkusativ-, Dativ- und Wechselpräpositionen** unterscheiden
- die richtige **Artikelendung** nachschlagen
- **Wo/Wohin** bei Wechselpräpositionen entscheiden
- Präpositionen in **Alltagssätzen** korrekt kombinieren

---

# 1. Übersicht

| Typ | Kasus | Frage |
| --- | --- | --- |
| Akkusativ | immer Akk. | durch, für, … |
| Dativ | immer Dat. | aus, bei, mit, … |
| Wechsel | Dat. oder Akk. | Wo? / Wohin? |

---

# 2. Akkusativpräpositionen (A1–A2)

| Präposition | Beispiel |
| --- | --- |
| **durch** | durch **den** Park |
| **für** | für **den** Kollegen |
| **gegen** | gegen **die** Wand |
| **ohne** | ohne **den** Zucker |
| **um** | um **die** Ecke |

**Zeit (Akk.):** am Montag, **um** drei Uhr, **für** eine Woche

---

# 3. Dativpräpositionen

| Präposition | Beispiel | Kontraktion |
| --- | --- | --- |
| **aus** | aus **der** Stadt | |
| **bei** | bei **dem** Arzt | **beim** |
| **mit** | mit **dem** Auto | |
| **nach** | nach Hause / nach Berlin | |
| **seit** | seit **einem** Jahr | |
| **von** | von **der** Firma | **vom** |
| **zu** | zu **der** Arbeit | **zur**, **zum** |

---

# 4. Wechselpräpositionen

| Präposition | Wo? (Dativ) | Wohin? (Akkusativ) |
| --- | --- | --- |
| **an** | an **der** Wand | an **die** Wand |
| **auf** | auf **dem** Tisch | auf **den** Tisch |
| **in** | in **der** Stadt | in **die** Stadt |
| **über** | über **dem** Tisch | über **den** Tisch |
| **unter** | unter **dem** Bett | unter **das** Bett |
| **vor** | vor **dem** Haus | vor **das** Haus |
| **hinter** | hinter **der** Tür | hinter **die** Tür |
| **neben** | neben **dem** Stuhl | neben **den** Stuhl |
| **zwischen** | zwischen **den** Häusern | zwischen **die** Häuser |

---

# 5. Entscheidungsbaum

```
Präposition bekannt?
├── durch/für/gegen/ohne/um → AKKUSATIV
├── aus/bei/mit/nach/seit/von/zu → DATIV
└── an/auf/in/über/unter/vor/hinter/neben/zwischen
    ├── Wo? → DATIV
    └── Wohin? → AKKUSATIV
```

---

# 6. Artikel im Dativ und Akkusativ

| Genus | Dativ | Akkusativ |
| --- | --- | --- |
| mask. | dem | den |
| fem. | der | die |
| neutr. | dem | das |
| Plural | den (+n) | die |

---

# 7. Häufige feste Ausdrücke

| Ausdruck | Kasus |
| --- | --- |
| zu Hause | Dativ (fest) |
| nach Hause | ohne Artikel |
| auf dem Land | Dativ (Wo?) |
| ins Kino | in + das (Akk.) |

---

# 8. Verb + Präposition (A2-Ausblick)

| Verb | Präposition | Beispiel |
| --- | --- | --- |
| sich freuen | auf + Akk. | sich freuen **auf** den Urlaub |
| sich interessieren | für + Akk. | sich interessieren **für** Musik |
| helfen | Dativ (ohne Präp.) | helfen **dem** Kind |
| denken | an + Akk. | denken **an** dich |

# 9. Übungen

## Aufgabe 1 — Kasus wählen

> Ich fahre mit _____ (der Bus).
> Ich lege das Buch auf _____ (der Tisch). (Wohin?)

Lösung:

> **dem** Bus — **den** Tisch

## Aufgabe 2 — Wo oder wohin?

> Das Bild hängt _____ (an) _____ (die Wand). (Position)

Lösung:

> **an der** Wand

## Aufgabe 3 — Akkusativpräposition

> Das Geschenk ist für _____ (die Mutter).

Lösung:

> **die Mutter**

## Aufgabe 4 — Kontraktion

> Ich gehe zu _____ (der Arzt).

Lösung:

> **zum** Arzt

## Aufgabe 5 — Mix

> Ich komme aus _____ (Vietnam). Ich wohne in _____ (die Stadt).

Lösung:

> **Vietnam** — **der** Stadt

Beschreibe dein Zimmer mit **fünf Wechselpräpositionen** (Wo? → Dativ).
