## Ziel dieses Kapitels

Dieses **Wörterbuch** listet die wichtigsten **Verb + Präposition** Paare für B1.

Am Ende kannst du:

- **50+ Verb-Präposition-Paare** nachschlagen
- **Kasus** pro Paar kennen
- Beispiele für **da-/wo-** bilden
- Paare **aktiv** in Sätzen üben

---

# 1. Akkusativ-Rektion

| Verb + Präp. | Beispiel |
| --- | --- |
| warten **auf** + Akk. | Ich warte **auf** dich. |
| sich freuen **auf** + Akk. | Wir freuen uns **auf** Montag. |
| sich freuen **über** + Akk. | Sie freut sich **über** das Geschenk. |
| denken **an** + Akk. | Denk **an** mich! |
| sich erinnern **an** + Akk. | Er erinnert sich **an** den Tag. |
| glauben **an** + Akk. | Ich glaube **an** dich. |
| sich interessieren **für** + Akk. | Interessierst du dich **für** Kunst? |
| sprechen **über** + Akk. | Wir sprechen **über** das Projekt. |
| diskutieren **über** + Akk. | Sie diskutieren **über** Kosten. |
| sich kümmern **um** + Akk. | Er kümmert sich **um** die Kinder. |
| bitten **um** + Akk. | Ich bitte **um** Hilfe. |
| sich vorbereiten **auf** + Akk. | Wir bereiten uns **auf** die Prüfung vor. |

# 2. Dativ-Rektion

| Verb + Präp. | Beispiel |
| --- | --- |
| abhängen **von** + Dat. | Es hängt **vom** Wetter ab. |
| sich bedanken **bei** + Dat. | Ich bedanke mich **bei** dir. |
| sich beschweren **bei** + Dat. | Er beschwert sich **beim** Chef. |
| gehören **zu** + Dat. | Das gehört **zum** Projekt. |
| helfen **bei** + Dat. | Sie hilft **bei** der Arbeit. |
| teilnehmen **an** + Dat. | Wir nehmen **am** Meeting teil. |
| zweifeln **an** + Dat. | Ich zweifle **am** Erfolg. |
| arbeiten **an** + Dat. | Er arbeitet **am** Bugfix. |

# 3. Wechsel / feste Verben

| Verb + Präp. | Beispiel |
| --- | --- |
| sich konzentrieren **auf** + Akk. | Konzentrier dich **auf** die Aufgabe! |
| achten **auf** + Akk. | Achte **auf** den Verkehr! |
| sich gewöhnen **an** + Akk. | Ich gewöhne mich **an** das Klima. |
| bestehen **aus** + Dat. | Das Team **besteht aus** fünf Leuten. |

---

# 4. da-/wo- mit Verben

> Ich **warte darauf**.
> **Worauf** **wartest** du?
> Wir **sprechen darüber**.
> **Wobei** **kann** ich **helfen**? → **bei** → **Wobei** hilfst du?

---

# 5. Lernstrategie

1. Verb + Präposition + **Kasus** als **ein Block**
2. **3 Beispielsätze** pro Paar
3. **da-/wo-** Variante üben

# 3. Übungen

## Aufgabe 1 — Paar finden

> Ich _____ mich _____ Musik. (interessieren)

Lösung:

> **interessiere** mich **für** Musik

## Aufgabe 2 — da-/wo-

> Wir sprechen über das Problem. → Wir sprechen _____

Lösung:

> **darüber**
