## Ziel dieses Kapitels

Diese Referenz hilft beim **Genus** (der/die/das) und bei **Pluralformen**.

Am Ende kannst du:

- **Suffixregeln** für Genus anwenden
- **häufige Pluralmuster** erkennen
- **Ausnahmen** und **Lernstrategien** kennen
- Nomen mit **Artikel** im Kontext lernen

---

# 1. Genus — Suffixregeln (Auswahl)

| Endung | oft | Beispiel |
| --- | --- | --- |
| -ung | die | die **Meinung**, die **Wohnung** |
| -heit, -keit | die | die **Freiheit**, die **Möglichkeit** |
| -schaft | die | die **Freundschaft** |
| -tion, -sion | die | die **Information** |
| -ität | die | die **Qualität** |
| -ik | die | die **Musik**, die **Logik** |
| -ei | die | die **Bäckerei** |
| -chen, -lein | das | das **Mädchen**, das **Männchen** |
| -ment | das | das **Dokument** |
| -um | das | das **Museum** |
| -er (Person/Tätigkeit) | der | der **Lehrer**, der **Computer** |
| -ismus | der | der **Tourismus** |

---

# 2. Plural — häufige Muster

| Muster | Beispiel |
| --- | --- |
| -e | der Tag → die **Tage** |
| -er (+ Umlaut) | das Kind → die **Kinder** |
| -en / -n | die Frau → die **Frauen** |
| -s | das Auto → die **Autos** |
| Umlaut only | der Vater → die **Väter** |
| kein Plural | die **Information** (auch: Informationen) |

---

# 3. Häufige B1-Nomen

| Nomen | Genus | Plural |
| --- | --- | --- |
| der Bericht | der | die Berichte |
| die Besprechung | die | die Besprechungen |
| das Projekt | das | die Projekte |
| die Erfahrung | die | die Erfahrungen |
| der Vorschlag | der | die Vorschläge |
| die Entscheidung | die | die Entscheidungen |
| das Ergebnis | das | die Ergebnisse |
| der Fehler | der | die Fehler |
| die Lösung | die | die Lösungen |
| der Kunde | der | die Kunden |

---

# 4. Strategien

1. **immer mit Artikel** lernen: **der** Bericht
2. **Suffixregeln** als Hinweis, nicht als Garantie
3. **Plural gleichzeitig** lernen
4. **Wortfamilien**: die **Entscheidung** (entscheiden)

---

# 5. Ausnahmen merken

> **der** See (See) vs. **die** See (Meer)
> **das** Band (Musik) vs. **der** Band (Gruppe)

---

# 3. Übungen

## Aufgabe 1 — Genus

> _____ Meinung, _____ Computer, _____ Mädchen

Lösung:

> **die** Meinung, **der** Computer, **das** Mädchen

## Aufgabe 2 — Plural

> der Bericht → _____

Lösung:

> **die Berichte**
