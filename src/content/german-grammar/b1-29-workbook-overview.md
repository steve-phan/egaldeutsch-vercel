## Ziel dieses Kapitels

Das **Arbeitsbuch** begleitet die Kapitel — so nutzt du es **effektiv**.

Am Ende kannst du:

- **Übungsarten** kennen (Lücken, Transformation, Produktion)
- **Lösungen** im Answer Key prüfen
- **Lernplan** mit Kapiteln verbinden
- **Fehlerheft** führen

---

# 1. Aufbau des Arbeitsbuchs

| Teil | Inhalt |
| --- | --- |
| Kapitelübungen | 1:1 zu Grammatikkapiteln |
| Review-Übungen | nach Einheit 1 und 2 |
| Mischübungen | mehrere Themen kombiniert |
| Tests | B1-Einstufungstest |

---

# 2. Übungstypen

| Typ | Ziel |
| --- | --- |
| **Lückentext** | Formen automatisch |
| **Transformation** | Aktiv→Passiv, direkt→indirekt |
| **Korrektur** | Fehler erkennen |
| **Produktion** | freies Schreiben |
| **Multiple Choice** | schnelle Prüfung |

---

# 3. So arbeitest du mit dem Buch

1. **Kapitel lesen** (Theorie + Beispiele)
2. **Übungen** ohne Lösung lösen
3. **Answer Key** (b1-30) nur **danach**
4. Fehler in **Fehlerheft** notieren
5. nach 3 Tagen **Fehler wiederholen**

---

# 4. Zeitplan — Beispiel (4 Wochen)

| Woche | Kapitel | Fokus |
| --- | --- | --- |
| 1 | 13–15 + b1-02–05 | Zeit + Genitiv + Relativ |
| 2 | 16–18 + b1-09–17 | Infinitiv + Passiv + Konjunktiv |
| 3 | b1-13–22 | Nebensätze + Anwendung |
| 4 | Review + Test | Prüfung |

---

# 5. Fehlerheft — Vorlage

| Datum | Fehler | Regel | Korrektur |
| --- | --- | --- | --- |
| … | *wegen dem* | wegen + Gen. | wegen des … |

# 6. Partner- und Selbstarbeit

- **Sprechübungen:** Relativsätze laut bilden
- **Schreibübungen:** E-Mail + Bericht pro Woche
- **Selbst:** Timer 20 Min. pro Übungssatz
