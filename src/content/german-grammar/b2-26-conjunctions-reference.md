## Ziel dieser Referenz

**Konnektoren** auf einen Blick — mit **Wortstellung** und **Komma**.

---

# 1. Koordinierende Konjunktionen

**Verb** bleibt auf Position 2 — **kein** Verb ans Ende

| Konjunktion | Bedeutung | Komma |
| --- | --- | --- |
| **und** | Addition | oft nein |
| **aber** | Kontrast | meist ja |
| **oder** | Alternative | je nach Kontext |
| **denn** | Begründung | meist ja |
| **sondern** | Korrektur | nach **nicht** |

> Ich **arbeite**, **aber** er **ruft** an.
> Nicht **heute**, **sondern** morgen.

---

# 2. Subordinierende Konjunktionen — Wahrnehmung

**Verb ans Ende** des Nebensatzes

| Konjunktion | Bedeutung |
| --- | --- |
| **weil** | Grund |
| **dass** | Inhalt |
| **wenn** | Bedingung/Zeit |
| **obwohl** | Konzession |
| **ob** | ob |
| **als** | Vergangenheit (einmalig) |

---

# 3. Subordinierende — Zeit

| Konjunktion | Bedeutung |
| --- | --- |
| **bevor** | bevor |
| **nachdem** | nachdem |
| **während** | während |
| **bis** | bis |
| **seit(dem)** | seit |

> **Nachdem** er **gegangen** **war**, **rief** ich an.

---

# 4. Subordinierende — B2-Erweiterung

| Konjunktion | Bedeutung |
| --- | --- |
| **indem** | dadurch dass |
| **falls** | falls |
| **ohne dass** | ohne dass |
| **als ob** | als ob |
| **damit** | damit (Zweck) |
| **sodass** | sodass (Folge) |

---

# 5. Zweiteilige Konnektoren

| Paar | Beispiel |
| --- | --- |
| **je … desto/umso** | **Je** mehr, **desto** besser |
| **nicht nur … sondern auch** | … |
| **entweder … oder** | … |
| **weder … noch** | … |
| **zwar … aber** | … |

---

# 6. Komma-Regel Kurzfassung

**Komma** vor **subordinierendem** Nebensatz (meistens)

**Kein** Komma vor **und/oder** (einfache Aufzählung)

# Übungen

## Aufgabe 1 — koordinierend oder subordinierend?

> weil

Lösung:

> **subordinierend** — Verb am Ende

## Aufgabe 2 — Satz vervollständigen

> Ich bleibe zu Hause, weil …

Lösung:

> … **weil** ich krank **bin**.

## Aufgabe 3 — je … desto

( viel / lernen ) → ( gut / werden )

Lösung:

> **Je** viel du **lernst**, **desto** besser **wirst** du.

## Aufgabe 4 — sondern

> Ich komme nicht heute, ___ morgen.

Lösung:

> … **sondern** morgen.
