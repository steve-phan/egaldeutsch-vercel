## Ziel dieses Kapitels

Das **B2-Arbeitsbuch** ergänzt die **Kapitel** mit **gezielten Übungen**.

---

# 1. Aufbau

| Teil | Inhalt |
| --- | --- |
| **A** | Wortstellung und Satzbau |
| **B** | Adjektive und Partizipien |
| **C** | Passiv |
| **D** | Konjunktiv und indirekte Rede |

---

# 2. Empfohlener Lernplan

**8 Wochen** à **3–4** Einheiten:

| Woche | Themen |
| --- | --- |
| 1–2 | Wortstellung (b2-02, b2-03, Kapitel 19) |
| 3 | Adjektive (Kapitel 20, b2-05) |
| 4 | Passiv (Kapitel 21, b2-07) |
| 5 | Konjunktiv (Kapitel 22–23, b2-09, b2-11) |
| 6 | Komplexe Sätze (Kapitel 24, b2-13–b2-16) |
| 7 | Review 1 + 2 |
| 8 | Niveautest |

---

# 3. Wie üben?

1. **Kapitel** lesen (15–20 Min.)
2. **Übungen** schreiben (10–15 Min.)
3. **Lösung** prüfen — **Fehler** notieren
4. **1** Satz **laut** sprechen
5. **1** Satz ins **Tagebuch** schreiben

---

# 4. Schreibheft

Führe ein **Übungsheft** mit:

- Datum
- Kapitel
- Fehlertyp
- Korrektur

---

# 5. Referenz-Kapitel

Nutze **b2-25** bis **b2-31** als **Nachschlagewerk** — nicht zum Auswendiglernen, sondern zum **Prüfen**.
