## Ziel dieser Referenz

**Funktionsverbgefüge (FVG)** verdichten Aussagen — diese Tabelle ist dein **Nachschlagewerk**.

---

# 1. Struktur

**Nomen + Funktionsverb** ≈ einfaches Verb

| Funktionsverb | Rolle |
| --- | --- |
| treffen, nehmen, leisten, üben | Handlung |
| stellen, bringen, gelangen | Zustandswechsel |
| kommen, gehen | Prozess |

---

# 2. Häufige FVG — Allgemein

| FVG | Einfaches Verb |
| --- | --- |
| eine **Entscheidung treffen** | **entscheiden** |
| eine **Prüfung vornehmen** | **prüfen** |
| **Kritik üben** | **kritisieren** |
| **Hilfe leisten** | **helfen** |
| **Abschied nehmen** | **sich verabschieden** |
| **in Frage stellen** | **bezweifeln** |
| **zur Verfügung stellen** | **bereitstellen** |
| **in Kraft treten** | **gelten** |
| **zur Anwendung kommen** | **angewendet werden** |
| **Einfluss nehmen auf** | **beeinflussen** |

---

# 3. FVG — Beruf und Verwaltung

| FVG | Kontext |
| --- | --- |
| **einen Antrag stellen** | Behörde |
| **Maßnahmen ergreifen** | Management |
| **Rücksicht nehmen auf** | Kooperation |
| **Kontakt aufnehmen mit** | Kommunikation |
| **Bedingungen schaffen für** | Planung |
| **Rechenschaft ablegen** | Verantwortung |
| **Auskunft erteilen** | Service |
| **Genehmigung erteilen** | Verwaltung |

---

# 4. FVG — Wissenschaft

| FVG | Bedeutung |
| --- | --- |
| **eine Analyse durchführen** | analysieren |
| **Schlussfolgerungen ziehen** | schließen |
| **Bezug nehmen auf** | sich beziehen |
| **in Betracht ziehen** | berücksichtigen |
| **zum Ausdruck bringen** | ausdrücken |
| **in Zweifel ziehen** | anzweifeln |

---

# 5. Präpositionen bei FVG

> **Rücksicht nehmen auf** + Akk.
> **Bezug nehmen auf** + Akk.
> **Einfluss nehmen auf** + Akk.
> **Antwort geben auf** + Akk.

---

# 6. Übungen

## Aufgabe 1

> Wir haben eine Entscheidung gemacht.

Lösung:

> Wir **haben** eine **Entscheidung getroffen**.

## Aufgabe 2

> Die Regel gilt ab Montag.

Lösung:

> Die Regel **tritt** ab Montag **in Kraft**.

## Aufgabe 3

> Man hat die These angezweifelt.

Lösung:

> Man **hat** die These **in Frage gestellt**.
