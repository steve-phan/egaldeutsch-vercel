## Ziel dieser Referenz

Schneller Überblick über **akademische Konnektoren** — nach Funktion sortiert.

---

# 1. Einleitung und Thema

| Konnektor | Beispiel |
| --- | --- |
| **Ausgehend von** | **Ausgehend von** der These … |
| **Im Rahmen von** | **Im Rahmen von** dieser Studie … |
| **Vor dem Hintergrund** | **Vor dem Hintergrund** der Krise … |
| **Hinsichtlich** | **Hinsichtlich** der Methodik … |

---

# 2. Argumentation

| Konnektor | Funktion |
| --- | --- |
| **Es lässt sich** behaupten | These |
| **Es spricht dafür, dass** | Pro-Argument |
| **Demgegenüber** | Gegenargument |
| **Ein wesentlicher Aspekt ist** | Gewichtung |
| **Nicht zuletzt** | Betonung |

# 7. Übungen

## Aufgabe 1

Verbinde mit passendem Konnektor: Die Daten sind begrenzt. Eine Aussage ist möglich.

Lösung:

> **Obwohl** die Daten begrenzt sind, ist eine Aussage möglich. / Die Daten sind begrenzt; **dennoch** ist eine Aussage möglich.

## Aufgabe 2

Schlussfolgerung: Die Hypothese ist falsch. Wir brauchen ein neues Modell.

Lösung:

> **Folglich** / **Somit** brauchen wir ein neues Modell.

Schreibe einen Essay-Absatz mit mindestens vier Konnektoren aus dieser Referenz.
