## Ziel dieses Leitfadens

**Register** = die richtige Grammatik für den **richtigen Kontext**.

---

# 1. Drei Ebenen

| Register | Merkmale | Typische Kontexte |
| --- | --- | --- |
| **informell** | du, Partikeln, Aktiv, kurz | Freunde, Familie |
| **neutral** | Sie/du je nach Kontext, klar | Alltag, viele Jobs |
| **formell** | Sie, Passiv, Nominal, FVG, Konjunktiv | Behörde, Wissenschaft, Vertrag |

---

# 2. Entscheidungstabelle

| Situation | Empfehlung |
| --- | --- |
| E-Mail an Behörde | **formell**: Passiv, **Sie**, keine Partikeln |
| Chat mit Team (Startup) | **neutral/informell**: du, **mal**, **doch** ok |
| wissenschaftlicher Artikel | **formell**: Nominal, Passiv, Konnektoren |
| Beschwerde beim Nachbarn | **neutral-höflich**: **Sie**, Konjunktiv II |
| Vertrag / AGB | **formell**: FVG, **ist/sind verpflichtet** |
| Standup (Tech) | **neutral**: Aktiv, Fachbegriffe, du |

---

# 3. Grammatik nach Register

| Struktur | informell | formell |
| --- | --- | --- |
| Bitte | Kannst du …? | **Könnten Sie** …? |
| Aussage | Das **ist doch** klar. | **Es** **lässt sich** feststellen, dass … |
| Prozess | Wir **machen** das. | Es **wird** **durchgeführt**. |
| Vermutung | Das **wird** wohl … | Es **dürfte** … **sein**. |
| Quelle | Man sagt … | **Dem Bericht zufolge** … |

---

# 4. Konjunktiv — wann?

| Form | Register | Funktion |
| --- | --- | --- |
| **Konjunktiv II** (könnte, wäre) | neutral–formell | Höflichkeit, Irrealis |
| **Konjunktiv I** | formell | indirekte Rede, Journalismus |
| Indikativ | informell–neutral | direkte Aussage |

---

# 5. Passiv — wann?

| Passiv ja | Passiv nein |
| --- | --- |
| Wissenschaft, Protokoll | persönliche E-Mail an Freund |
| Agens unbekannt/unwichtig | klare Verantwortung zeigen |
| formelle Distanz | dynamische Erzählung |

---

# 6. Partikeln — wann?

**Ja:** Gespräch, lockere E-Mail (du), Erklärung für Laien
**Nein:** Behördenbrief, wissenschaftlicher Text, Vertrag

---

# 7. Übungen

## Aufgabe 1

Kontext: E-Mail an das Finanzamt. Formuliere: Ich brauche eine Kopie.

Lösung:

> **Ich bitte** um **Übersendung** einer Kopie. / **Könnten Sie** mir eine Kopie **zusenden**?

## Aufgabe 2

Kontext: Slack an Kollegen (du). Formuliere formell → informell.

Lösung:

> **Kannst du** mir **mal** den Link **schicken**?

## Aufgabe 3

Ordne zu: Passiv / Partikel / FVG — welcher Kontext?

Lösung:

> Passiv: Laborbericht / Partikel: Kaffee mit Freund / FVG: Geschäftsbericht
