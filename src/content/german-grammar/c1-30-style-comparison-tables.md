## Ziel dieser Referenz

**Dieselbe Idee** — drei Register. Welche Grammatik passt?

---

# 1. Allgemeine Aussage

| gesprochen | schriftlich neutral | akademisch |
| --- | --- | --- |
| Das **ist doch** klar! | Das **ist** klar. | **Es lässt sich** feststellen, dass … |
| **Ich glaub**, das stimmt. | **Ich denke**, das stimmt. | **Es** **dürfte** zutreffen, dass … |
| **Man** sagt, … | **Laut** Angaben … | **Dem Bericht zufolge** … |

---

# 2. Bitte und Aufforderung

| gesprochen | schriftlich neutral | akademisch |
| --- | --- | --- |
| **Mach mal** bitte! | **Bitte** erledigen Sie das. | **Es wird** gebeten, … **zu** … |
| **Kannst du** …? | **Könnten Sie** …? | **Es** **empfiehlt** sich, … |
| **Gib her!** | **Bitte** senden Sie … | **Die Übersendung** wird **erbeten**. |

---

# 3. Prozess beschreiben

| gesprochen | schriftlich neutral | akademisch |
| --- | --- | --- |
| Wir **machen** das gerade. | Wir **arbeiten** daran. | Es **wird** daran **gearbeitet**. |
| **Die** Daten **wurden** geprüft. | **Wir haben** geprüft. | **Die Prüfung** der Daten **erfolgte** … |
| **Er** **hat** **gefixt**. | **Der Fehler wurde** behoben. | **Eine Behebung** des Fehlers **fand** statt. |

---

# 4. Vermutung

| gesprochen | schriftlich neutral | akademisch |
| --- | --- | --- |
| **Der ist** wohl krank. | Er **ist** wahrscheinlich krank. | Er **dürfte** krank **sein**. |
| **Kann nicht** stimmen! | Das **kann nicht** stimmen. | Dies **lässt sich** **ausschließen**. |
| **Angeblich** … | **Er soll** … **haben**. | **Mutmaßlich** … / **Es wird** **berichtet**, … |

---

# 5. Kontrast und Folgerung

| gesprochen | schriftlich neutral | akademisch |
| --- | --- | --- |
| **Aber** …! | **Jedoch** … | **Demgegenüber** … |
| **Trotzdem**! | **Dennoch** … | **Nichtsdestotrotz** … |
| **Also** … | **Deshalb** … | **Folglich** / **Somit** … |

# 6. Tech / Software

| Slack (du) | E-Mail (Sie) | Dokumentation |
| --- | --- | --- |
| **Deploy** **ist** **down**. | **Das Deployment** **ist** **ausgefallen**. | **Es kam** zu einem **Ausfall** des **Deployments**. |

# 7. Übungen

## Aufgabe 1

Formuliere in allen drei Registern: Die Ergebnisse überraschen.

Lösung:

> gesprochen: **Die Ergebnisse** **sind ja** **krass**! / neutral: Die Ergebnisse **sind überraschend**. / akademisch: **Die Ergebnisse** **sind** **überraschend** **und** **werfen** **neue** Fragen **auf**.

## Aufgabe 2

Welches Register: **Es wird hiermit mitgeteilt, dass …**

Lösung:

> **akademisch / formell** (Behörde, offiziell)

## Aufgabe 3

Schreibe denselben Satz in **gesprochen** und **akademisch** (je 1 Satz).
