## Ziel dieser Referenz

**Modalpartikeln** ändern den **Ton**, nicht die Kernbedeutung — Übersicht für C1.

---

# 1. Hauptpartikeln

| Partikel | Kernnuance | Beispiel |
| --- | --- | --- |
| **doch** | Betonung, Einladung, Erwartung | Komm **doch** mit! |
| **mal** | weicher, casual | Schau **mal**! |
| **halt** | Resignation, Faktizität | So **halt**. |
| **eben** | unvermeidlich | Das **eben** akzeptieren. |
| **ja** | Bekanntes betonen | Das ist **ja** toll! |
| **wohl** | Vermutung | Er ist **wohl** krank. |
| **denn** | Neugier (Frage) | Was machst du **denn**? |
| **ruhig** | Erlaubnis | Frag **ruhig**. |
| **schon** | Beruhigung | Das **schon** machen. |
| **bloß/nur** | Warnung | Pass **bloß** auf! |

---

# 2. doch — Varianten

| Kontext | Wirkung |
| --- | --- |
| Imperativ | freundlicher | **Warte doch**! |
| Aussage | „wie erwartet" | Das **doch** klar! |
| Widerspruch | **aber doch** | Das stimmt **doch**! |
| Konjunktion | **aber** (Satzanfang) | **Doch** er kam. |

---

# 3. ja — Varianten

| Kontext | Beispiel |
| --- | --- |
| Überraschung | **Ja** **sogar** er war da! |
| Bekanntes | Du **ja** **kennst** das! |
| Warnung | Das **ja** **nicht** vergessen! |

**Nicht** verwechseln mit **ja** (Antwort) = „yes"

---

# 4. wohl vs. Modalverb

| Konstruktion | Bedeutung |
| --- | --- |
| Er **ist wohl** krank. | Vermutung (Partikel) |
| Er **muss wohl** krank sein. | abgeschwächte Vermutung |
| **Das wird** wohl **klappen**. | vorsichtige Prognose |

---

# 5. Register — wann Partikeln?

| Register | Partikeln |
| --- | --- |
| Freunde, Familie | **ja** |
| Kollegen (du) | **mal**, **doch** ok |
| formelle E-Mail | **nein** |
| wissenschaftlicher Text | **nein** |
| Vertrag / AGB | **nein** |

---

# 6. Kombinationen

> **Doch mal** schauen! (locker)
> **Halt eben** so. (Resignation)
> **Ja wohl** nicht! (empört)

**C1:** Kombinationen verstärken **Ton** — nicht in formeller Schrift.

---

# 7. Häufige Fehler

| Fehler | Korrektur |
| --- | --- |
| *Es wird doch mitgeteilt* | **Hiermit wird** mitgeteilt (formell) |
| Partikel wörtlich übersetzen | Ton verstehen, nicht 1:1 |
| zu viele Partikeln | max. 1–2 pro Satz |

---

# 8. Übungen

## Aufgabe 1

Was drückt **halt** aus?

> So **halt**.

Lösung:

> **Resignation** — „so ist es nun mal" / Faktizität akzeptieren.

## Aufgabe 2

doch oder ja? ___ das ist interessant!

Lösung:

> **ja** (Betonung des Bekannten/Überraschenden)

## Aufgabe 3

Formuliere um: formell statt *Komm doch mal vorbei!*

Lösung:

> **Besuchen Sie** uns **gerne**. / **Wir** **würden** uns **freuen**, wenn Sie **vorbeikommen**.

## Aufgabe 4

Erkläre den Ton von: **Das schon**, aber **halt** nicht heute.

Lösung:

> **schon** = einräumen; **halt** = Einschränkung/resignativ — „grundsätzlich ja, aber eben nicht heute".
