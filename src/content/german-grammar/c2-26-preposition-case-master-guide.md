## Ziel dieses Kapitels

**Präposition + Kasus** ist auf C2 **eine Einheit** — dieser Leitfaden bündelt **alle** Major-Gruppen.

Am Ende kannst du:

- **Akkusativ-, Dativ-, Genitiv-** und **Wechselpräpositionen** **sicher** wählen
- **feste Verbindungen** mit **Kasus** abrufen
- **registerbezogene** Varianten (**wegen** + Gen./Dat.) **kennen**
- **Zeit-, Ort-, Modalpräpositionen** **systematisch** nutzen

---

# 1. Nur Akkusativ

| Präp. | Beispiel |
| --- | --- |
| **durch** | **durch** **den** **Park** |
| **für** | **für** **das** **Team** |
| **gegen** | **gegen** **den** **Wind** |
| **ohne** | **ohne** **Fehler** |
| **um** | **um** **die** **Ecke**; **um** **acht** **Uhr** |

---

# 2. Nur Dativ

| Präp. | Beispiel |
| --- | --- |
| **aus** | **aus** **Berlin** |
| **bei** | **bei** **der** **Arbeit** |
| **mit** | **mit** **dem** **Auto** |
| **nach** | **nach** **Hause** |
| **seit** | **seit** **2020** |
| **von** | **von** **mir** |
| **zu** | **zu** **dem** **Arzt** |
| **gegenüber** | **mir** **gegenüber** |

# 3. Wechselpräpositionen (Wo? / Wohin?)

| Präp. | Wo? (Dat.) | Wohin? (Akk.) |
| --- | --- | --- |
| **an** | **an** **der** **Wand** | **an** **die** **Wand** |
| **auf** | **auf** **dem** **Tisch** | **auf** **den** **Tisch** |
| **in** | **in** **der** **Stadt** | **in** **die** **Stadt** |
| **über** | **über** **dem** **Tisch** | **über** **den** **Tisch** |
| **unter** | **unter** **dem** **Tisch** | **unter** **den** **Tisch** |
| **vor** | **vor** **dem** **Haus** | **vor** **das** **Haus** |
| **hinter** | **hinter** **dem** **Haus** | **hinter** **das** **Haus** |
| **neben** | **neben** **dem** **Haus** | **neben** **das** **Haus** |
| **zwischen** | **zwischen** **den** **Häusern** | **zwischen** **die** **Häuser** |

---

# 4. Genitivpräpositionen (formell)

| Präp. | Beispiel |
| --- | --- |
| **während** | **während** **des** **Meetings** |
| **wegen** | **wegen** **des** **Wetters** |
| **trotz** | **trotz** **des** **Problems** |
| **innerhalb** | **innerhalb** **von** **24** **h** |
| **außerhalb** | **außerhalb** **der** **SLA** |
| **statt** | **statt** **des** **Altsystems** |

**Umgangssprachlich:** **wegen** **dem** — **nicht** in formellen Texten.

---

# 5. Wechsel + Genitiv (selten)

**während, trotz, wegen** — oft **Genitiv** in **Schrift**.

---

# 6. Zeit — subtile Paare

| Ausdruck | Beispiel |
| --- | --- |
| **am** | **am** Montag, **am** Morgen |
| **im** | **im** März, **im** Jahr 2024 |
| **um** | **um** acht (punktuell) |
| **gegen** | **gegen** acht (ungefähr) |
| **von … bis** | **von** **9** **bis** **17** **Uhr** |
| **ab** | **ab** **morgen** |

---

# 7. Feste Verbindungen + Kasus

| Verbindung | Kasus |
| --- | --- |
| **in Frage stellen** | Akk. |
| **zum Schluss kommen** | Dat. |
| **im Gegensatz zu** | Dat. |
| **angesichts** | Gen. |
| **im Hinblick auf** | Akk. |
| **aufgrund** | Gen. |

---

# 8. von / aus / bei / zu

| Präp. | Nuance |
| --- | --- |
| **von** | Person, Ausgangspunkt |
| **aus** | Material, Herkunft (Stadt) |
| **bei** | Ort/Anwesenheit |
| **zu** | Richtung (Person/Veranstaltung) |

---

# 9. Kontraktionen

> **ins**, **zum**, **beim**, **vom**, **am**, **im**, **zur**

**Register:** Kontraktionen **neutral** — **aufs** umgangssprachlicher.

---

# 10. Deutsch für Software Engineers

| Kontext | Beispiel |
| --- | --- |
| Deployment | **in** **die** **Produktion** |
| Laufzeit | **während** **des** **Deployments** |
| Abhängigkeit | **abhängig** **von** |
| Compliance | **gemäß** **der** **DSGVO** |

---

# 11. Häufige Fehler

Falsch: *in Montag* → **am** Montag
Falsch: *nach Hause zu* → **nach** **Hause**
Falsch: *wegen dem* (formell) → **wegen** **des**

---

# 12. Übungen

## Aufgabe 1 — Wechselpräposition

> Das Buch liegt _____ dem Tisch. (Wo?)

Lösung:

> **auf** **dem** Tisch

## Aufgabe 2 — Genitiv

> _____ des Sturms wurde das Event abgesagt.

Lösung:

> **Wegen** / **Aufgrund**

## Aufgabe 3 — Zeit

> Wir starten _____ acht Uhr _____ Punkt.

Lösung:

> **um** acht Uhr

## Aufgabe 4 — von/aus

> Sie kommt _____ Wien. (Herkunft)

Lösung:

> **aus** Wien

## Aufgabe 5 — Feste Verbindung

> Wir müssen das _____ Frage stellen.

Lösung:

> **in** Frage stellen
