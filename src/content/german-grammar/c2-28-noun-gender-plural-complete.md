## Ziel dieses Kapitels

**Genus** und **Plural** sind **kein Zufall** — diese Referenz systematisiert **Regeln**, **Ausnahmen** und **Strategien**.

Am Ende kannst du:

- **Suffixe** für **Genus-Vorhersage** nutzen
- **Pluralmuster** **erkennen**
- **Ausnahmen** und **Fremdwörter** **einordnen**
- **Compound-Nomen** **analysieren**

---

# 1. Genus — Übersicht

| Genus | Häufigkeit (grob) |
| --- | --- |
| **der** (mask.) | ~45 % |
| **die** (fem.) | ~35 % |
| **das** (neutr.) | ~20 % |

**C2:** Regeln = **Wahrscheinlichkeit**, nicht **Garantie**.

---

# 2. Maskulin — typische Suffixe

| Suffix | Beispiel |
| --- | --- |
| **-er** | **der** **Lehrer**, **der** **Computer** |
| **-ling** | **der** **Schützling** |
| **-ig** | **der** **König** |
| **-ismus** | **der** **Kapitalismus** |
| **-or** | **der** **Doktor** |
| **-ant** | **der** **Elefant** |

**Ausnahme:** **das** **Fenster**, **das** **Zimmer** (-er neutr.)

---

# 3. Feminin — typische Suffixe

| Suffix | Beispiel |
| --- | --- |
| **-ung** | **die** **Wohnung** |
| **-heit/-keit** | **die** **Freiheit**, **die** **Möglichkeit** |
| **-schaft** | **die** **Freundschaft** |
| **-tion/-sion** | **die** **Nation**, **die** **Diskussion** |
| **-ei** | **die** **Bäckerei** |
| **-in** | **die** **Lehrerin** (Personen fem.) |
| **-ur** | **die** **Kultur** |

---

# 4. Neutrum — typische Suffixe

| Suffix | Beispiel |
| --- | --- |
| **-chen/-lein** | **das** **Mädchen**, **das** **Büchlein** |
| **-ment** | **das** **Dokument** |
| **-um** | **das** **Museum** |
| **-tum** | **das** **Eigentum** |
| **-nis** | **das** **Ergebnis** |
| **Ge-** | **das** **Gespräch** |

---

# 5. Plural — Muster

| Muster | Beispiel |
| --- | --- |
| **-e** | **der** **Tag** → **die** **Tage** |
| **-er** + Umlaut | **das** **Kind** → **die** **Kinder** |
| **-en/-n** | **der** **Name** → **die** **Namen** |
| **-s** | **das** **Auto** → **die** **Autos** |
| **-ien** | **das** **Studium** → **die** **Studien** |
| **unverändert** | **der** **Lehrer** → **die** **Lehrer** |
| **Singular = Plural** | **das** **Deer** → **die** **Deer** (selten) |

---

# 6. Fremdwörter

| Endung | oft | Beispiel |
| --- | --- | --- |
| **-e** (fem.) | die | **die** **Lounge** |
| **-o** (neut.) | das | **das** **Konto** |
| **-ma** (neut.) | das | **das** **Drama** |

**Anglizismen:** oft **das** — **das** **Backend**, **das** **Ticket**

---

# 7. Compound-Nomen

> **das** **Bundes** + **ministerium** → **das** **Bundesministerium**

**Genus** = **letztes** Glied (**ministerium** → **das**)

---

# 8. N-Deklination (Plural -en)

> **der** **Student** → **die** **Studenten**
> **der** **Mensch** → **die** **Menschen**

---

# 9. Lernstrategien

| Strategie | Aktion |
| --- | --- |
| **Mit Artikel** lernen | nicht *Tisch*, sondern **der** **Tisch** |
| **Suffix-Regeln** | bei **neuen** Wörtern anwenden |
| **Gruppen** | thematische Listen (Technik, Alltag) |
| **Fehlerlog** | persönliche **Ausnahmen** sammeln |

---

# 10. Deutsch für Software Engineers

| Nomen | Genus | Plural |
| --- | --- | --- |
| **der** **Server** | mask. | **die** **Server** |
| **die** **Datenbank** | fem. | **die** **Datenbanken** |
| **das** **Framework** | neutr. | **die** **Frameworks** |
| **der** **Bug** | mask. | **die** **Bugs** |
| **das** **Deployment** | neutr. | **die** **Deployments** |

---

# 11. Häufige Fehler

Falsch: *die Problem* → **das** **Problem**
Falsch: *der Information* → **die** **Information**
Falsch: Plural von **das** **Auto** → **die** **Autos** (nicht *Autoe*)

# 12. Übungen

## Aufgabe 1 — Genus

> _____ Universität (Suffix -ität)

Lösung:

> **die** Universität

## Aufgabe 2 — Plural

> der Tag → Plural

Lösung:

> **die** **Tage**

## Aufgabe 3 — Compound

> das Haupt + das Gebäude

Lösung:

> **das** **Hauptgebäude**

## Aufgabe 4 — Fremdwort

> _____ Backend

Lösung:

> **das** Backend

## Aufgabe 5 — Strategie

> Nenne drei Nomen aus deinem Fachgebiet — mit Artikel und Plural.

Lösung:

> Individuell — z. B. **der** **Microservice**, **die** **Microservices**
