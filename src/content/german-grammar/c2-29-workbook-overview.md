## Ziel dieses Kapitels

Die **Roadmap** umfasst **Kapitel** + **Übungen** + **Lösungen** — dieses Kapitel erklärt das **Gesamtsystem**.

Am Ende kannst du:

- die **Struktur** des Übungsbuchs **überblicken**
- **Lernzyklen** (lesen → üben → prüfen) **planen**
- **Review-Einheiten** und **Tests** **einordnen**
- den **Lösungsschlüssel** (c2-30) **effektiv** nutzen

# 3. Empfohlener Lernzyklus

```
1. Kapitel lesen (15–30 Min.)
2. Übungen ohne Lösung (10–20 Min.)
3. Lösung prüfen — Fehler notieren
4. Schwachstelle → Referenzkapitel
5. Nach 5 Kapiteln: Review-Einheit
```

# 8. Spiral-Lernen

Themen **kehren** zurück:

> Kasus A1 → B1 Genitiv → C2 Nuance
> Passiv B1 → B2 → C2 Stil
> Konjunktiv B1 → B2 → C2 Schattierungen

**Altes** Kapitel **mit neuem** Blick **wiederholen**.
