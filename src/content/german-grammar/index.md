# Die vollständige deutsche Grammatik: Von A1 bis C2

**The Complete German Grammar Roadmap** — German Grammar for Everyday Life, Work, Study and Professional Communication.

This book is the grammar sister project for EgalDeutsch: a structured, mobile-friendly German grammar roadmap from **A1 to C2**.

It is not a translation of an English grammar book. German needs its own path: gender, cases, article endings, sentence brackets, separable verbs, subordinate clauses, passive, Konjunktiv I/II, particles, register and high-level style.

## How to Use This Book

1. Start with your current CEFR level.
2. Read the context first, not the table.
3. Speak the examples aloud.
4. Do the controlled practice.
5. Open the solution only after trying.
6. Revisit the same grammar in Alltag, Beruf and Software contexts.

## Book Architecture

The full roadmap contains **186 published chapters** across six CEFR levels, plus reference appendices, review units and tests.

- **A1 Foundations** (31): sentence basics, gender, articles, present tense, accusative, questions.
- **A2 Everyday Grammar** (31): dative, two-way prepositions, Perfekt, separable verbs, subordinate clauses.
- **B1 Independent Grammar** (31): past tenses, genitive, relative clauses, infinitives, passive, Konjunktiv II.
- **B2 Upper-Intermediate Grammar** (31): advanced word order, adjective declension, passive alternatives, indirect speech.
- **C1 Advanced Grammar** (31): subjective modals, nominalisation, academic cohesion, register and editing.
- **C2 Mastery Grammar** (31): subtle grammar choices, sophisticated syntax, rhetoric and ambiguity control.

## The Chapter Pattern

Every chapter follows the same learning rhythm: goal, context, pattern discovery, form, meaning, word-order diagram, tables, common errors, English comparison, Vietnamese learner notes, everyday German, work German, software-engineer German, exercises, transformation, correction, speaking, writing and review.

Solutions are collapsible so mobile reading stays calm and focused.

## Spiral Curriculum

Important structures return at higher levels — passive, cases, adjective endings, word order, modal verbs, subordinate clauses, relative clauses, Konjunktiv and nominalisation each deepen from A2 through C2.
