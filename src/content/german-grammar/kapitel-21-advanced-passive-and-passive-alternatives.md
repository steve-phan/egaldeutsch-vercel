## Ziel dieses Kapitels

Auf B2 beherrschst du Passiv **in der Tiefe** und kennst **Alternativen**, wenn Passiv unnatürlich klingt.

Am Ende kannst du:

- Passiv in **Futur, Plusquamperfekt** und mit **Modalverben** bilden
- **Zustandspassiv** (sein + Part. II) vom **Vorgangspassiv** trennen
- **man**-Sätze und **sich lassen** als Alternativen nutzen
- **Passivsätze** stilistisch bewusst wählen

---

# 1. Passiv-Zeitformen — Überblick

| Zeit | Vorgangspassiv |
| --- | --- |
| Präsens | wird gemacht |
| Präteritum | wurde gemacht |
| Perfekt | ist gemacht **worden** |
| Plusquamperfekt | war gemacht **worden** |
| Futur I | wird gemacht **werden** |

> Der Fehler **war** schon **behoben worden**, bevor wir testeten.
> Die Lösung **wird** morgen **implementiert werden**.

---

# 2. Passiv mit Modalverben

> Der Bericht **muss** noch **geschrieben werden**.
> Das Problem **kann** **gelöst werden**.
> Die Daten **dürfen** nicht **gelöscht werden**.

**Klammer:** Modal + Partizip II + **werden**

Im Perfekt:

> Der Fix **hätte** früher **deployed werden sollen**.

---

# 3. Zustandspassiv — sein + Partizip II

| Vorgangspassiv | Zustandspassiv |
| --- | --- |
| Die Tür **wird** geöffnet. | Die Tür **ist** geöffnet. |
| Der Server **wird** neu gestartet. | Der Server **ist** neu gestartet. |

**Vorgang:** Prozess — **werden**
**Zustand:** Ergebnis — **sein**

> Das Ticket **ist** **bearbeitet**. (Zustand: fertig)
> Das Ticket **wird** **bearbeitet**. (Prozess: gerade)

---

# 4. Passiv-Alternativen

## man + Aktiv

> **Man** hat den Fehler **behoben**.
> Hier **arbeitet man** bis 18 Uhr.

## sich lassen + Infinitiv

> Das Problem **lässt sich** **lösen**.
> Der Text **lässt sich** gut **lesen**.

**Bedeutung:** es ist möglich / es kann getan werden

## -bar / -lich Adjektive

> **lösbar**, **lesbar**, **verfügbar**
> Das Problem **ist lösbar**.

---

# 5. Wann welche Konstruktion?

| Situation | Bevorzugt |
| --- | --- |
| formeller Bericht | Passiv |
| Prozess läuft | Vorgangspassiv |
| Ergebnis/Zustand | Zustandspassiv |
| allgemeine Aussage | **man** |
| Möglichkeit | **sich lassen** |

---

# 6. Agent mit von/durch

| Präposition | Typisch für |
| --- | --- |
| **von** | Personen, Abteilungen |
| **durch** | Mittel, Ursache, Technik |

> Der Fix **wurde von** dem Team **entwickelt**.
> Der Schaden **wurde durch** einen Bug **verursacht**.

---

# 7. Häufige Fehler

Falsch: *Der Server ist repariert worden.* (wenn Prozess gemeint)
Richtig: Der Server **wird/wurde** repariert. / Der Server **ist** repariert. (Zustand)

Falsch: *Das muss gelöst sein werden.*
Richtig: Das **muss gelöst werden**.

Falsch: *sich lässt zu lösen*
Richtig: sich **lässt lösen**

---

# 8. Vergleich mit Englisch

| Englisch | Deutsch |
| --- | --- |
| It **can be** solved | Es **lässt sich lösen** / Es **kann gelöst werden** |
| The door **is** open | Die Tür **ist geöffnet** (Zustand) |
| One **fixes** the bug | **Man** behebt den Bug |

---

# 9. Fehler für vietnamesischsprachige Lernende

- **worden** vs. **geworden** im Passiv-Perfekt
- Zustandspassiv nicht mit Vorgang verwechseln
- **sich lassen** braucht **sich**

---

# 10. Deutsch im Beruf

> Die Maßnahmen **werden** derzeit **umgesetzt**.
> Der Vertrag **ist** **unterzeichnet**.
> **Man** erwartet eine Entscheidung bis Freitag.
> Das **lässt sich** noch **ändern**.

---

# 11. Deutsch für Software Engineers

> Der Bug **wurde durch** ein fehlendes Null-Check **verursacht**.
> Das Release **hätte** gestern **deployed werden müssen**.
> Die API **ist** **dokumentiert**.
> Das Problem **lässt sich** mit einem Patch **beheben**.

# 12. Übungen

## Aufgabe 1 — Vorgangs- oder Zustandspassiv?

> Die Tür _____ (schließen). — Sie ist zu.

Lösung:

> Die Tür **ist geschlossen** (Zustand). / Jemand **schließt** die Tür (Aktiv).

## Aufgabe 2 — Plusquamperfekt Passiv

> Der Fix _____ (deployen), bevor der Test startete.

Lösung:

> Der Fix **war deployed worden**, bevor der Test startete.

## Aufgabe 3 — Modal + Passiv

> Man muss die Logs prüfen.

Lösung:

> Die Logs **müssen geprüft werden**.

## Aufgabe 4 — sich lassen

> Das Problem kann leicht gelöst werden.

Lösung:

> Das Problem **lässt sich** leicht **lösen**.

## Aufgabe 5 — Fehler korrigieren

> Der Fehler ist durch das Team behoben worden. Das lässt zu ändern.

Mögliche Lösung:

> Der Fehler **wurde durch** das Team **behoben**. Das **lässt sich** ändern.

## Aufgabe 6 — Schreibaufgabe

Schreibe einen Incident-Report (6 Sätze): mindestens 2× Passiv, 1× Zustandspassiv, 1× sich lassen.
