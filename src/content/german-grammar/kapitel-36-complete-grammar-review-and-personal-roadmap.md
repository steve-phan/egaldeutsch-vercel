# 1. Das deutsche Grammatik-System — ein Bild

```
A1: Satz + Genus + Artikel + Präsens + Akkusativ + Fragen
         ↓
A2: Dativ + Wechselpräp. + Perfekt + Trennbar/Reflexiv + Adj. + Nebensätze
         ↓
B1: Vergangenheit + Genitiv + Relativ + Infinitiv + Passiv + Konjunktiv II
         ↓
B2: Wortstellung + Adj. voll + Passiv+ + Konj. I/II + Komplexe Sätze
         ↓
C1: Modals subj. + Partizipien + Nominal/FVG + Akademisch + Register + Lektorat
         ↓
C2: Nuance + Satzarchitektur + Evidentialität + Rhetorik + Meister-Edit
```

**Spirale:** Themen wie **Passiv, Kasus, Wortstellung, Konjunktiv** kehren **immer** mit höherer Tiefe zurück.
